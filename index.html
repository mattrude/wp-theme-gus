mattrude.github.com / The Gus Theme

The Gus theme for WordPress will be the future theme for http://mattrude.com.

Features

  • Photo Gallery Support
  • Gallery index page
  • EXIF Support for images
  • Ability to tag people in images
  • Custom taxonomy (people, place, event)
  • Custom people gallery page
  • Threaded comments

Wanted Features

  • Twiter Custom Post Type.

Prerequisites

  • WordPress 3.3+

Support

If you are having issues using the Gus theme, please file a bug on this projects github Issues page.
